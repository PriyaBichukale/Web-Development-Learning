The Cascading Algorithm

  1. Position and order of appearance(Which is recently given that will apply)
  2. Specificity
    1. Inline CSS (1st priority)
    2. ID selector (2nd pri)
    3. Class or Attribute Selector (3rd)
    4. Element Selector (4th)
    5. Universal Selector (5th)
  3. Origin (If not given any style User Agent Stylesheet will apply)
  4. Important (!important) (Ever 1st priority)
